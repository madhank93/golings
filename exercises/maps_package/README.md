# The maps Package

The `maps` package (Go 1.21+) provides generic helpers for whole-map operations
— cloning, comparing, bulk deletion, and iterating keys and values — so you stop
hand-rolling loops for common tasks. (This is the standard-library package, not
the `map` literal syntax covered in the `maps` exercises.)

- **mapspkg1** — `maps.Clone` for an independent copy (assignment shares!)
- **mapspkg2** — `maps.DeleteFunc` deletes matching entries in one pass
