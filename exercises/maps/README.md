# Maps

A map is an unordered collection of key/value pairs: `map[K]V`. Create with
`make` or a literal, read with `m[k]`, and use the comma-ok form `v, ok := m[k]`
to tell a missing key from a zero value. Delete with `delete(m, k)`.
