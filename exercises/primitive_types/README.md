# Primitive types

Go's built-in types: booleans, numeric types (`int`, `int8…64`, `uint…`,
`float32/64`, `complex`), `byte` (alias for `uint8`), `rune` (alias for `int32`),
and `string`. Conversions between numeric types must be explicit.
