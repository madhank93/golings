## What is golings?

golings is a set of small, broken-on-purpose Go exercises you fix one at a time —
[rustlings](https://rustlings.rust-lang.org/) for Go. An interactive terminal UI
runs each exercise as you save and walks you from variables to concurrency.

<CardGrid>
  <Card title="97 exercises" icon="puzzle">
    32 topics across a beginner → advanced track, current through Go 1.26.
  </Card>
  <Card title="Instant feedback" icon="rocket">
    A Bubble Tea TUI re-runs the exercise on save; tests **and** golangci-lint
    must pass before it advances.
  </Card>
  <Card title="Covers the essentials" icon="open-book">

  </Card>

## Start in two commands

```sh
git clone https://github.com/madhank93/golings && cd golings
mise run watch
```

See [Getting started](/getting-started/) for the full setup.
